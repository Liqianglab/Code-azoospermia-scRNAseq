Supplementary Figure 8 code

Files:
- supfig8_source_data.py
  Builds panel-wise source-data tables, raw support copy, file mapping, availability table,
  and panel-source_code-confidence checklist files for Supplementary Figure 8 (A-E).

- supfig8_plot_panels.py
  Replots check figures for Panels A-E from generated source data.
  Outputs to: ./supfig8_plots/

Run order:
1) python3 source data/supfig8_code/supfig8_source_data.py
2) python3 source data/supfig8_code/supfig8_plot_panels.py
